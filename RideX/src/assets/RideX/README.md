# RideX Project

## Overview
RideX is a web application designed to facilitate the booking of test rides for various vehicles, including electric and petrol options. The application provides users with a seamless experience to explore vehicle options, schedule test rides, and manage their profiles.

## Features
- User authentication (Login and Registration)
- Main dashboard for user navigation
- Detailed information on electric and petrol vehicles
- Countdown timer for upcoming events
- Search functionality with suggestions
- Booking system for test rides

## Getting Started
To get started with the RideX application, clone the repository and install the necessary dependencies:

```bash
git clone <repository-url>
cd RideX
npm install
```

## Usage
After installing the dependencies, you can start the application using:

```bash
npm start
```

This will launch the application in your default web browser.
